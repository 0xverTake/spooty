Pour ajouter une vidéo de fond :

1. Téléchargez une vidéo de plage paradisiaque depuis Pixabay ou Pexels
2. Renommez-la en "beach.mp4"
3. Placez-la dans ce dossier

Spécifications recommandées :
- Résolution : 1920x1080 ou 4K
- Format : MP4
- Durée : 10-30 secondes en boucle
- Contenu : Plage, palmiers, coucher de soleil
- Taille de fichier : < 10MB (compressé pour le web)

Sources gratuites suggérées :
- https://pixabay.com/videos/
- https://www.pexels.com/videos/
- https://mixkit.co/free-stock-video/
